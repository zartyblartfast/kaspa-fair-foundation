# Kaspa Fair Foundation

This is the clean, reusable foundation repository for Kaspa Fair.

- `kaspa-fair-lab` remains the historical lab/evidence repository.
- Current proven live path is **Testnet-10 (TN10)**.
- Mainnet is disabled and not supported.
- Roulette is future PoC/demo work and is not implemented yet.
- No secrets should be committed in this repository.
